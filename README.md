# ⛺ ElonCamp - Solusi Sewa Alat Camping Terlengkap

Aplikasi penyewaan alat outdoor berbasis web dinamis (PHP Native). Project ini dikembangkan sebagai tugas UAS Pemrograman Web dengan semangat inovasi dan kecintaan terhadap alam.

## 🏔️ Tentang ElonCamp
Nama **ElonCamp** diambil dari nama salah satu anggota tim kami, **Elon**, yang merupakan seorang pecinta gunung sejati. Aplikasi ini bertujuan untuk mendigitalisasi proses persewaan alat gunung agar lebih praktis, transparan, dan efisien.

## 🚀 Fitur Utama
- **Katalog Alat:** Menampilkan daftar perlengkapan camping (tenda, nesting, carrier, dll).
- **Sistem Sewa & Stok:** Manajemen stok otomatis saat barang disewa.
- **Hitung Otomatis:** Perhitungan biaya sewa dan denda keterlambatan (Problem Solver).
- **Dashboard Admin:** Pengelolaan produk dan validasi transaksi.

## 🛠️ Stack Teknologi
- **Bahasa:** PHP Native
- **Database:** MySQL (MariaDB)
- **Desain:** CSS & Bootstrap 5
- **Version Control:** Git & GitHub

## 📂 Struktur Folder Project
- `/assets` : File CSS, JS, dan Gambar Produk.
- `/config` : Koneksi database (`database.php`).
- `/admin` : Halaman kelola data untuk admin.
- `index.php` : Halaman utama (Landing Page).
- `README.md` : Dokumentasi project.
